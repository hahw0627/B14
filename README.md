# 건슬링어: 미래의 수호자

## 프로젝트 개요
**건슬링어: 미래의 수호자**는 미래 지구를 배경으로 하는 2D 방치형 게임입니다. 플레이어는 서부의 전설적인 총잡이가 되어 괴물들에 맞서 싸우며, 다양한 무기와 능력을 통해 캐릭터를 성장시켜 나갑니다.

## 주요 기능
- **자동 전투**: 게임이 자동으로 진행되는 자동 전투 시스템.
- **다양한 무기**: 각기 다른 특성을 가진 무기와 업그레이드.
- **스킬 시스템**: 강력한 스킬을 통해 적을 제압.
